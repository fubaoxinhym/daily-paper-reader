## 摘要
无人机在执行任务时越来越依赖视觉定位能力，以便从复杂空中场景中的多样化指令中定位任务相关目标。然而，现有的指代表达理解基准和方法大多基于纯文本查询和单一目标输出，这就限制了它们在涉及参考图像、多模态指令、目标缺失以及多个有效目标实例等实际无人机场景中的适用性。为弥补这一不足，我们提出了“通用指代”这一广义无人机指代任务，它同时扩展了查询模态和输出基数。我们构建了UniRef-UAV多模态基准，支持纯文本、纯图像以及文本+图像查询，且目标基数取决于模态：纯文本和文本+图像查询支持无目标、单目标和多目标定位，而纯图像查询侧重于存在感知的单实例定位。该基准还为视觉查询的泛化提供了域内和跨域评估协议。我们进一步提出了UAV-URNet基线，这是一种检测风格的模型，将异质查询映射到共享查询空间，并通过集合预测预测可变大小的目标集合。大量实验表明，与大型通用多模态大语言模型相比，UAV-URNet提供了稳定且可复现的基线，具有更一致的无目标判别能力以及更轻量、更易复现的实现。额外的域分析、查询表示分析和消融研究表明，多模态查询有助于减少视觉查询歧义，并促进更统一的查询-目标对齐空间。注释、视觉查询裁剪图/图像、训练/验证/测试划分、评估脚本和基线代码将公开提供，以促进可复现研究。

## Abstract
Unmanned aerial vehicles (UAVs) increasingly rely on visual grounding capabilities to localize task-relevant targets from diverse instructions in complex aerial scenes. Existing referring expression comprehension (REC) benchmarks and methods, however, are largely built around text-only queries and single-object outputs, which limits their applicability to practical UAV scenarios involving reference images, multimodal instructions, absent targets, and multiple valid target instances. To address this gap, we introduce \emph{Universal Referring}, a generalized UAV referring task that jointly expands the query modality and the output cardinality. We construct \emph{UniRef-UAV}, a multimodal benchmark that supports text-only, image-only, and text+image queries with modality-dependent target cardinality, where text-only and text+image queries admit no-target, single-target, and multi-target grounding while image-only queries focus on existence-aware single-instance grounding. It also provides in-domain and cross-domain evaluation protocols for visual-query generalization. We further present \emph{UAV-URNet}, a detection-style baseline that maps heterogeneous queries into a shared query space and predicts variable-size target sets through set prediction. Extensive experiments show that UAV-URNet provides a stable and reproducible baseline with more consistent no-target discrimination and a more lightweight, reproducible implementation than large general-purpose MLLMs. Additional domain analysis, query-representation analysis, and ablation studies demonstrate that multimodal queries help reduce visual-query ambiguity and promote a more unified query--target alignment space. The annotations, visual query crops/images, train/validation/test splits, evaluation scripts, and baseline code will be made publicly available to facilitate reproducible research.

## 1. 核心问题与整体含义（研究动机和背景）
无人机在安全巡逻、搜索救援、交通监控等应用中需要将任务指令（如文本描述、参考图像）精准定位到空中复杂场景中的具体视觉目标。然而，现有的指代表达理解（REC）基准和方法普遍存在两大局限：① 查询模态单一（仅支持文本），② 输出假定为单个目标。这无法适应实际无人机场景中出现的多模态指令（如同时使用文本和图像引用）、目标缺失（所查询物体不在当前视野）以及多个有效目标并存等复杂情况。为此，论文提出了 **“通用指代”（Universal Referring）** 任务，旨在同时扩展查询模态和输出基数，使其更贴近真实无人机操作需求，并构建了对应的多模态基准 **UniRef-UAV** 和基线模型 **UAV-URNet**。

## 2. 方法论
### 2.1 核心思想
将指代理解统一为模态无关的语义定位问题：给定场景图像 \(I\) 和查询 \(Q\)（文本、图像或文本+图像），模型输出一个可变大小的边界框集 \(B\)，其合法基数取决于查询模态：
- 文本/文本+图像：输出可以是 0、1 或多个框
- 图像：只能输出 0 或 1 个框（存在感知单实例定位）

### 2.2 关键技术细节
- **统一查询表示**：基于 GroundingDINO 的“2+2”检测式架构，通过不同方式将异质查询映射到共享的查询嵌入空间：
  - 文本查询：直接使用语言编码器输出。
  - 图像查询：使用与场景图相同的视觉编码器，再通过轻量 MLP 投影到文本嵌入维度，作为伪文本 token。
  - 文本+图像查询：将视觉查询特征通过交叉注意力层增强文本表示，生成多模态查询 token。
- **集合预测**：保留 GroundingDINO 的解码器，通过查询条件候选匹配输出可变数目边界框。
- **训练策略**：冻结预训练的视觉和文本编码器，仅优化跨模态交互模块、投影 MLP 和检测解码器，降低训练成本。

### 2.3 评估指标
- **Precision@(F1=1, IoU≥0.5)**：仅当预测框与真值框的匹配集合达到完全正确（F1=1）且 IoU≥0.5 时才计为正确。
- **N-acc**：对无目标样本，要求模型输出空集才计为正确。

## 6. 主要结论与发现
1. **现有方法不足**：在 UniRef-UAV 上，即便是最强通用 MLLM（Qwen3.6-27B）在文本设置下也仅达到约 20% 的 Precision，而面向定位的 MLLM 表现更差，表明现有方法难以同时处理多模态查询、可变基数和无目标场景。
2. **UAV-URNet 优势**：作为检测式基线，在无目标判别（N-acc > 90%）上远超所有 MLLM，且实现轻量、可复现。在正样本定位上仍有提升空间。
3. **多模态查询互补**：加入文本可显著提升图像查询的 N-acc（从 0.299 到 0.840）；去除图像会严重降低文本+图像性能，说明视觉和文本信息互补且各有不可替代性。
4. **域内 vs 跨域**：域内视觉查询定位更好，跨域仍是挑战，增加文本有助于抑制假阳性。
5. **公共嵌入空间**：通过中心化相似性分析，验证了 UAV-URNet 能将不同模态映射到可比较的共享空间。

## 7. 优点
- **任务定义新颖**：首次在 UAV 场景中提出支持多模态查询和可变基数的通用指代，贴近真实应用。
- **基准构建扎实**：从 22 个数据集整合、人工标注、严格质量控制，提供域内/跨域设置。
- **基线设计简洁可扩展**：基于成熟的 GroundingDINO，通过最少改动支持多模态输入，易于复现和修改。
- **实验全面**：对比了 20+ 种方法，涵盖多种架构；消融实验设计直接，结论清晰。
- **开源承诺**：代码、数据、评估脚本均计划公开，促进社区研究。
- **已提交IEEE TMM**，质量有保证。

## 8. 不足与局限
- **UAV-URNet 定位精度有限**：在图像和文本+图像设置下的 Precision 约 0.55-0.69，低于某些 MLLM（如 Qwen3.5-0.8B 在图像设置达 0.772），但在无目标判别上显著领先。
- **跨域泛化仍弱**：跨域视觉查询的 N-acc 较低（图像设置仅 0.259），说明模型对域外视觉原型匹配不稳定。
- **小目标定位困难**：论文提到精细定位小目标仍是挑战，但未专门优化。
- **硬无目标案例**：部分无目标样本语义上易混淆，模型存在过度预测倾向。
- **部署限制**：未在真实无人机平台上测试，实际推理速度、内存、能耗等未报告。
- **基准规模有限**：156k 查询相比自然场景 GREC（如 gRefCOCO 258k）略小，但仍属合理范围。
