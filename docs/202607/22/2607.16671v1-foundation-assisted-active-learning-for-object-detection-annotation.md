## 摘要
遥感目标检测的标注成本高昂，而现有的主动学习方法在目标检测场景中仍面临定位与分类不确定性耦合、冷启动阶段严重定位噪声以及高召回候选提议导致的伪多样性等挑战。为解决这些问题，我们提出了一种基础模型协同的主动学习与半自动标注框架，用于高效构建遥感目标检测数据集。我们构建了基于UPN+SAM2的参考定位源（SA-source）与检测器预测源（OD-source）的双源机制，并进一步提出基础模型增强的双源不确定性估计方法，通过联合建模定位一致性与分类置信度来提升冷启动阶段的样本选择质量。此外，我们提出目标中心多样性采样，利用DINOv2特征与SAM2掩码构建目标级表征，以在抑制伪多样性的同时提升样本覆盖度。为应对半自动标注阶段的几何噪声，我们设计了双源框切换策略，用SA-source中匹配的精细化框替代噪声检测框，从而降低人工框调整负担。在DIOR、HRSC2016、DOTAv2和FAIR1M上的实验表明，本方法在大多数标注预算下取得优于或相当的结果，且在低预算场景下冷启动样本效率显著更强。

## Abstract
The annotation cost for remote sensing object detection is high, while existing active learning methods still face several challenges in object detection scenarios, including the coupling of localization and classification uncertainty, severe localization noise in the cold-start stage, and pseudo-diversity caused by high-recall candidate proposals. To address these issues, we propose a foundation-model-collaborative active learning and semi-automatic annotation framework for efficient construction of remote sensing object detection datasets. We build a dual-source mechanism consisting of a reference localization source (SA-source) based on UPN+SAM2 and a detector prediction source (OD-source), and further propose a Foundation-model-enhanced Dual-Source Uncertainty estimation to improve sample selection quality in the cold-start stage by jointly modeling localization consistency and classification confidence. Furthermore, we propose Object-Centric Diversity Sampling, which constructs object-level representations using DINOv2 features and SAM2 masks to improve sample coverage while suppressing pseudo-diversity. To address geometric noise in the semi-automatic annotation stage, we design Dual-Source Box Switching, which replaces noisy detector boxes with matched refined boxes from the SA-source, thereby reducing the manual burden of box refinement. Experiments on DIOR, HRSC2016, DOTAv2, and FAIR1M show that our method achieves superior or comparable results under most annotation budgets, with notably stronger cold-start sample efficiency in the low-budget regime.

# 基于基础模型辅助的主动学习目标检测标注论文总结

## 1. 核心问题与整体含义（研究动机与背景）

遥感目标检测依赖大规模高质量标注数据，但手动标注成本极高（尤其面对密集小目标、尺度变化、复杂成像等）。主动学习作为缓解标注负担的有效范式，通过优先选择信息量大的样本进行标注，以有限预算最大化检测性能。然而，现有主动学习方法在目标检测场景中仍存在三大挑战：

- **分类与定位不确定性的耦合**：传统方法多依赖检测器置信度，难以区分分类不确定性与定位不确定性，导致冷启动阶段样本选择受不稳定边界框的严重干扰。
- **冷启动阶段的严重定位噪声**：初始标注很少，检测器分类能力可能因预训练而快速改善，但定位仍高度不稳定，基于单一检测置信度的采样策略失效。
- **高召回候选提议导致的伪多样性**：经典多样性采样在高召回候选（如全景分割/通用提议网络产出）中容易受重叠碎片和冗余提议影响，降低标注预算利用效率。

为此，本文提出**基础模型协同的主动学习与半自动标注框架**，将视觉基础模型（UPN、SAM2、DINOv2）深度集成到主动学习流水线中，旨在实现遥感目标检测数据集的高效构建。

## 2. 方法论

本文方法的**核心思想**：构建一个双源协作机制——“SA-source”（基于UPN→SAM2的参考定位源）和“OD-source”（当前检测器的预测源），在此基础上设计两阶段样本选择（不确定性过滤 + 多样性采样），并引入冷启动半自动标注辅助机制。

### 关键技术细节

#### 2.1 SA-source 预计算（离线一次性）

- **UPN 生成类别无关提议框**：对每一张未标注图像，UPN 输出高召回率提议框及目标分数。
- **SAM2 精细化**：以每个 UPN 框作为提示，SAM2 生成对象掩码，再提取紧密边界框（tight box），获得稳定的几何先验。
- **缓存复用**：所有结果（边界框、掩码、目标分数）仅计算一次，后续各轮主动学习直接读取，避免重复推理。

#### 2.2 基础模型增强的双源不确定性估计（FDSU）

- **双源定义**：SA-source 提供离线缓存的稳定参考框；OD-source 提供当前检测器的预测框和分类置信度。
- **匈牙利匹配**：忽略类别标签，仅基于 IoU 代价进行一对一定位对齐，得到匹配对集合。
- **定位一致性与分类置信度融合**：对每个匹配对，计算定位一致性分数 σ_match，并与分类置信度 σ_od 通过**调和均值**融合为对象级不确定性 μ_jk：
  \[
  \mu_{jk} = 1 - \frac{2 \sigma_{od}^k \sigma_{match}^{jk}}{\sigma_{od}^k + \sigma_{match}^{jk} + \epsilon}
  \]
- **图像级不确定性聚合**：采用 Top-K 聚合（选取图像内不确定性最高的 K 个对象均值），最终按不确定性排序筛选出候选集 I^{(s)}_{cond}（扩张因子 θ > 1）。

#### 2.3 对象中心多样性采样（OCDS）

- **对象嵌入**：对候选图像中的 SA-source 候选对象，利用 SAM2 掩码在 DINOv2 特征图上进行掩码引导池化，得到归一化对象特征。
- **类原型构建**：在已标注集上对每个类别使用 k-means++ 聚类出 p 个原型，代表已覆盖的外观。
- **新颖性得分**：对候选对象，计算其特征与其预测类原型的最小余弦距离作为 novel。然后 Top-K_d 聚合为图像级覆盖得分 Cov(I)。
- **碎片抑制**：针对 UPN 可能产生的过碎片化，引入四项惩罚：小面积碎片比例、空间重叠冗余（IoU）、语义冗余（特征余弦）、候选数量对数；最终多样性得分 D(I) = Cov(I) - λ Frag(I)。
- 从候选集中选出 Top-n 最高 D(I) 的图像作为本轮标注集。

#### 2.4 双源框切换（DSBS）——冷启动半自动标注

- **动机**：冷启动阶段检测器定位质量差，若直接展示检测框给标注员，将增加大量框调整工作。
- **操作**：对当前标注图像，将已匹配的 OD 框替换为对应 SA 框（几何来自 SA，类别标签来自 OD）；仅当 IoU≥τ_sw 时才切换，避免无关替换；未匹配的 OD 框保留；未匹配的 SA 框可可选展示为“待确认”。
- **动态调度**：可基于固定轮次或自适应（全局匹配质量阈值）决定何时停止 DSBS，避免后期干扰。

## 6. 论文的主要结论与发现

- 提出的 FDSU 能更准确地评价样本信息量，尤其在冷启动阶段显著优于仅依赖检测器置信度的传统不确定性方法。
