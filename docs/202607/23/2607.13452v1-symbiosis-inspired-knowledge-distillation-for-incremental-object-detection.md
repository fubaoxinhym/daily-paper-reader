## 摘要
增量目标检测（IOD）旨在将检测器扩展到新类别，同时保留先前获得的知识。现有方法通常采用类增量学习视角，分离特征空间以锐化决策边界。然而，这种分离导向的范式可能忽略检测中的对象共生现象，其中共现和遮挡引入了空间和语义依赖关系，而这些依赖关系得益于共享表示。忽略这些依赖关系会扭曲共享表示，加剧新旧类别之间的混淆，并加速灾难性遗忘。为了解决这个问题，我们提出了共生启发的知识蒸馏（SIKD），它在两个互补层面上显式利用对象共生。空间共生蒸馏（SpSD）关注旧模型对新任务中对象具有高重叠响应的共生区域。它保留可泛化的旧类别线索，抑制类别特定偏差和冗余，并通过槽对齐监督将精炼后的证据蒸馏到匹配空间位置的新模型中。语义共生蒸馏（SeSD）通过为旧类别形成置信加权原型并调整它们在旧类别logits上的类间软排名来维持类别级结构，从而在适应过程中稳定语义拓扑。大量实验证明了所提方法的有效性和优越性。

## Abstract
Incremental object detection (IOD) aims to extend detectors to new categories while retaining previously acquired knowledge. Existing methods often adopt a class incremental learning perspective, separating feature spaces to sharpen decision boundaries. However, this separation-oriented paradigm may overlook object symbiosis in detection, where co-occurrence and occlusion introduce spatial and semantic dependencies that benefit from shared representations. Ignoring these dependencies distorts the shared representations, exacerbates confusion between old and new classes, and accelerates catastrophic forgetting. To address this, we propose Symbiosis-Inspired Knowledge Distillation (SIKD), which explicitly leverages object symbiosis at two complementary levels. Spatial Symbiosis Distillation (SpSD) focuses on symbiotic regions where the old model responds with high overlap to objects in the new task. It preserves generalizable old class cues, suppresses class-specific bias and redundancy, and distills the refined evidence to the new model at matched spatial locations with slot-aligned supervision. Semantic Symbiosis Distillation (SeSD) maintains class level structure by forming confidence weighted prototypes for old classes and aligning their inter class soft ranks over the old class logits, which stabilizes the semantic topology during adaptation. Extensive experiments demonstrate the effectiveness and superiority of the proposed method.

- **问题**：增量目标检测（Incremental Object Detection, IOD）要求在持续学习新类别的同时，不遗忘已学过的旧类别。现有方法多采用类增量学习的分离导向范式，即通过隔离特征空间来锐化决策边界。
- **背景**：这种分离导向的范式忽略了检测中固有的对象共生现象——对象之间的共现和遮挡会引入空间和语义依赖关系，而这些依赖关系需要共享的特征表示才能被有效建模。
- **动机**：忽视共生依赖会导致共享表示扭曲，加剧新旧类别间的混淆，从而加速灾难性遗忘（catastrophic forgetting）。因此，论文提出显式利用对象共生关系进行知识蒸馏，以缓解遗忘并提升检测性能。

## 2. 方法论：核心思想与关键技术细节

### 核心思想
- 提出 **共生启发的知识蒸馏（Symbiosis-Inspired Knowledge Distillation, SIKD）**，在空间和语义两个互补层面上显式利用对象共生，将旧模型中的可泛化知识高效迁移至新模型。

### 关键技术细节

#### （1）空间共生蒸馏（Spatial Symbiosis Distillation, SpSD）
- **目的**：保留旧类别中可泛化的空间线索，抑制类别特定的偏差和冗余。
- **实现**：
  - 识别旧模型在新任务图像中产生高重叠响应的“共生区域”（即旧模型对旧类对象与新类对象共现区域均有响应）。
  - 通过**槽对齐监督（slot-aligned supervision）**，将这些区域中精炼后的证据（refined evidence）蒸馏到新模型对应空间位置。
  - 该过程同时压制旧模型中对新任务无用的类别特异噪声，使得蒸馏特征更具通用性。

#### （2）语义共生蒸馏（Semantic Symbiosis Distillation, SeSD）
- **目的**：维持类别级的语义结构，防止新任务学习扭曲旧类别的语义拓扑。
- **实现**：
  - 为每个旧类别构建**置信加权原型（confidence-weighted prototypes）**，综合不同样本的激活强度。
  - 针对这些原型在旧类别logits上的**类间软排序（inter-class soft ranks）**进行对齐，即保持新旧模型对旧类别之间相对排序的一致。
  - 这种方法能稳固语义拓扑，使新模型在适应新类别时不破坏旧类别间的相对关系。

### 算法流程说明
1. 加载预训练的旧模型，冻结部分参数。
2. 对新任务数据，同时通过旧模型和新模型进行前向传播。
3. SpSD：计算旧模型在特征图上的响应重叠区域，提取旧模型对这些区域的响应值，经过精炼（如抑制低置信度位置）后，通过槽对齐损失迫使新模型在相同空间位置产生相似响应。
4. SeSD：从旧模型提取特征并生成置信加权原型；计算原型在旧类别logits上的排序；通过排序对齐损失（如KL散度或边际排序损失）使新模型的旧类别logits排序与旧模型一致。
